## Getting Started

Create a new folder 

```shell
npm init
```
This command creates a package.json file.

install nodemon
```shell
npm install nodemon -D
```

update scripts section in package.json
```shell
"scripts": { 
  "test": "echo \"Error: no test specified\" && exit 1", 
  "start": "nodemon server" 
},
```

start our server with this command
```shell
npm start
```
